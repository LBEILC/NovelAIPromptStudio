## Design Context

### Users

NovelAI creators who import generated images, recover prompts and generation metadata, organize reusable tags and Vibe references, and iterate on prompt versions during long desktop sessions. The product must remain equally usable on Windows and macOS.

### Brand Personality

Precise, calm, and studio-grade. The interface should feel like a focused creative workbench: capable and technical without becoming cryptic, decorative, or visually noisy.

### Aesthetic Direction

A refined Lobe UI asset studio with slate neutrals, a restrained blue primary color, quiet surfaces, and image-first composition. Use the locally bundled Geist, Geist Mono, and HarmonyOS Sans SC families consistently across platforms. Preserve the dense professional layout, but never achieve density by making text microscopic. Avoid neon sci-fi styling, glassmorphism, generic dashboard cards, ornamental motion, and one-sided accent strips for persistent selection states.

### Design Principles

1. Readability before density: primary working text is comfortable at normal viewing distance; captions are never smaller than 0.6875rem (11px).
2. Preserve the artwork as the visual focus while keeping prompt controls easy to scan and operate.
3. Use a small semantic type scale with consistent roles, weights, line heights, and tabular numerals for parameters.
4. Respect platform conventions for menus, title bars, shortcuts, paths, dialogs, and secure storage.
5. Prefer quiet, explicit state feedback over animation or decorative effects.
6. Express persistent selection with a complete tinted surface or full border. Reserve directional indicators for transient drag-and-drop placement only.
